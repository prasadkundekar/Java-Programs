## 🛠 How to Run the Programs

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/java-programs.git
   cd java-programs
   ```

2. **Compile a Java program**

   ```bash
   javac ProgramName.java
   ```

3. **Run the program**

   ```bash
   java ProgramName
   ```
